# AgentMatrix 任务追踪

> 版本：v0.2  
> 日期：2026-07-16  
> 状态：本轮完成，后续增强待排期  
> 范围：管理页底部 Agent-Skill 关系矩阵

## 目标定位

关系矩阵在 v0.2 中不再作为主流程组件，而是作为“高级诊断”入口保留。默认折叠，仅用于快速横向查看多个 skill 在多个 agent 下的状态，并联动打开对应 skill 详情。

矩阵暂不承担批量执行职责。批量修复统一进入 `BatchSyncPanel`，避免矩阵和批量面板出现两套操作模型。

## 当前问题

- [x] 矩阵默认折叠，已从主流程空间降级。
- [x] 单元格点击语义不清：当前发出 `action` 事件，但父组件未处理。
- [x] 状态口径与 `useSkillAgentStatus` / 批量面板未完全统一。
- [x] 单元格只靠颜色，不够自解释。
- [x] tooltip 信息不足，缺少 agent、状态、source path、symlink target。
- [x] 不应提供会修改文件系统的直接操作入口。
- [x] 文案需要明确“高级诊断”定位。
